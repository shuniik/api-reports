# Reportes para POS
1. Clonar el repositorio
2. ``` yarn install ```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno con los datos reales del servidor
5. Levantar modo desarrollo ```yarn start:dev```
